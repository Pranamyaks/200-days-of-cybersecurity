# Day 07 - Network Layer (OSI Model)

## What is Network Layer?
Network Layer is the 3rd layer of the OSI model.  
It is responsible for routing data from source to destination across networks.

## Main Functions

1. Logical Addressing  
   - Uses IP address to identify devices  

2. Routing  
   - Finds the best path for data transmission  

3. Packet Forwarding  
   - Sends data packets from one network to another  

## Protocols

- IP (Internet Protocol)  
- ICMP (Internet Control Message Protocol)

## Example
When you send data over the Internet,  
the network layer decides the path it will take.

## 🔐 Cybersecurity Relevance
- Used in IP tracking and filtering  
- Important for understanding attacks like IP spoofing
