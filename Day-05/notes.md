# Day 04 - Presentation Layer (OSI Model)

## What is Presentation Layer?
Presentation Layer is the 6th layer of the OSI model.  
It is responsible for formatting, encrypting, and compressing data.

## Main Functions

1. Data Translation  
   - Converts data into a readable format  
   - Example: ASCII, Unicode

2. Encryption 
   - Secures data before sending  
   - Example: SSL/TLS

3. Compression  
   - Reduces data size for faster transmission  

## Example
When you open a secure website (HTTPS),  
the presentation layer encrypts the data.

## Cybersecurity Relevance
- Protects sensitive data using encryption  
- Helps prevent data theft and attacks
