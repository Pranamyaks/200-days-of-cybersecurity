# Day 05 - Session Layer (OSI Model)

##  What is Session Layer?
Session Layer is the 5th layer of the OSI model.  
It manages and controls communication sessions between devices.

##  Main Functions

1. Session Establishment  
   - Starts the connection between sender and receiver  

2. Session Management  
   - Maintains and controls the connection  

3. Session Termination  
   - Ends the connection after communication  

##  Example
When you log in to a website,  
the session layer keeps you logged in until you log out.

##  Cybersecurity Relevance
- Manages user sessions  
- Prevents unauthorized access  
- Important for authentication systems
