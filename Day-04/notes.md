# OSI Model (Open Systems Interconnection Model)

The OSI Model is a 7-layer conceptual framework used to understand how data is transmitted over a network. It divides the communication process into layers, where each layer performs a specific function, enabling standardization and interoperability between different systems.

## Key Points:
- Consists of 7 layers: Application, Presentation, Session, Transport, Network, Data Link, Physical
- Each layer has a specific role in data communication
- Helps in troubleshooting and designing networks
- Ensures compatibility between different devices and protocols
# Application Layer (OSI Layer 7)

- Top layer of OSI model
- Provides network services to user applications

## Functions:
- Web browsing (HTTP/HTTPS)
- Domain resolution (DNS)
- Email services (SMTP, IMAP, POP3)
- File transfer (FTP)

## Key Points:
- Closest to user
- Creates requests and processes responses
- Does not handle IP, MAC, or routing

## Example:
User enters www.google.com → DNS resolves IP → HTTP request sent → server responds with webpage
