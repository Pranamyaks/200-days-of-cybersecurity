# Day 08 - Data Link Layer & Physical Layer (OSI Model)

## Data Link Layer (Layer 2)
Data Link Layer is responsible for node-to-node communication within the same network.

### Functions
- Uses MAC address for communication  
- Error detection  
- Frame formation  

### Example
Switch uses MAC address to send data to the correct device.

## Physical Layer (Layer 1)
Physical Layer is responsible for transmitting raw bits over a physical medium.

### Functions
- Converts data into signals  
- Handles cables, wires, and hardware  

###  Example
Ethernet cables, signals, electrical transmission

##  Key Point
Data Link = MAC + Frames  
Physical = Signals + Hardware  

## 🔐 Cybersecurity Relevance
- Data Link: MAC spoofing attacks  
- Physical: Physical access can lead to security breaches
