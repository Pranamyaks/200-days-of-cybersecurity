# 📡 Networking Devices – Short Definitions

## 🔴 Hub
A hub is a networking device that connects multiple devices in a network and broadcasts data to all connected devices without filtering.

## 🟡 Modem
A modem is a device that converts digital signals to analog and vice versa to connect a network to the internet.

## 📡 Access Point (AP)
An access point is a device that connects to a wired network and provides wireless (Wi-Fi) connectivity to multiple devices.
