# Day 09 - IP Address, TCP vs UDP, TCP 3-Way Handshake

## What is IP Address?
An IP (Internet Protocol) Address is a unique identifier for a device on a network.

### Types:
- IPv4 (e.g., 192.168.1.1)
- IPv6 (long format)

##  TCP vs UDP

### TCP (Transmission Control Protocol)
- Connection-oriented  
- Reliable (guarantees delivery)  
- Slower  

### UDP (User Datagram Protocol)
- Connectionless  
- Faster  
- No guarantee of delivery  

##  Key Difference
TCP = Reliable  
UDP = Fast  

## TCP 3-Way Handshake

It is the process to establish a connection between client and server.

### Steps:
1. SYN → Client sends request  
2. SYN-ACK → Server responds  
3. ACK → Client confirms  

Connection is established after this.

##  Cybersecurity Relevance
- IP address used in tracking and filtering  
- TCP/UDP used in network attacks  
- Handshake is important for detecting suspicious connections
